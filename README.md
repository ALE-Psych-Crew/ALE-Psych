**ALE Psych** is a *rewrite* of [**Psych Engine**](https://github.com/ALE-Psych-Crew/ALE-Psych), focused on **SOFTCODING**, **customization**, and **adding more capabilities** to users through both **scripts** and **JSON**

> [!NOTE]
> This is a WIP, but even so, you can make complete mods there. All you need is to have the Psych Engine Editors (preferably 0.7.3) handy. We will soon be working on new editors for the project to make it possible to take advantage of the new features

##### Our goal before was simple: when this project was a fork of Psych, we simply wanted to eliminate the need for compilation, but maintaining the Psych Engine code was difficult, and it wasn't fully organized or optimized. Due to this and other problems, we decided to redo the project, but while maintaining the essence and convenience of Psych

---

## Main Differences Compared to Psych Engine

1. Support for Custom Menus and Submenus
2. Support for Unlimited Characters
3. Support for Unlimited StrumLines
4. Extensive customization via JSON and Scripts
5. More readable and organized code
6. Significant optimizations
7. A completely new PlayState (to facilitate gameplay, effects, and customization)
8. Custom classes
9. Lua modules
10. Improvements in scripting for greater stability and flexibility

> [!NOTE]
> Whenever you can use scripts, you will have the option to choose between Lua, HScript, or both!
