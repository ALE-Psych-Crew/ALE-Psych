# LuaWindowsAPI

### > `showMessageBox(caption:String, message:String, ?icon:#if, WINDOWS_API, MessageBoxIcon, #else, Int, #end)`

#### Display a text box from another window 

- `caption`: Window name 

- `message`: Text box content 

- `icon`: Text box type 

---

### > `setWindowBorderColor(r:Int, g:Int, b:Int)`

#### Applies a color to the window border 

- `r`: Red 

- `g`: Green 

- `b`: Blue 

---

### > `clearTerminal()`

#### Cleans the game terminal/console 

---

### > `showConsole()`

#### Shows the game terminal/console 

---
