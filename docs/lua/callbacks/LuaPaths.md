# LuaPaths

### > `precacheSound(file:String, ?missingPrint:Bool)`

#### Preloads a sound 

- `file`: Path of the audio 

- `missingPrint`: Defines whether to show a warning if the file does not exist 

---

### > `precacheMusic(file:String, ?missingPrint:Bool)`

#### Preloads music 

- `file`: Path of the audio 

- `missingPrint`: Defines whether to show a warning if the file does not exist
